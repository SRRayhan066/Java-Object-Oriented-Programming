Q. What is method overloading?
Ans: If a class has multiple methods having same name but different in parameters, it is known as Method Overloading.

Q. Why method overloading is not possible by changing the return type only?
Ans: With the same number of argument,same type of argument and different type of return type occurs compilation error.

Q. Can we overload java main method?
Ans: Yes,But JVM will run the main method which receives string array as arguments.

Q. What is method overriding?
Ans: If subclass (child class) has the same method as declared in the parent class, it is known as method overriding in Java.
